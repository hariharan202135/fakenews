# 📰 Fake News Detection System

This project detects whether a news article is REAL or FAKE using Machine Learning.

## 🚀 Features
- Classifies news using AI
- Uses TF-IDF and Naive Bayes
- Simple web interface using Streamlit

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## ⚙️ How It Works
1. Input news text
2. Convert text using TF-IDF
3. Model predicts Real or Fake
4. Display result with confidence

## ▶️ How to Run
pip install pandas numpy scikit-learn streamlit  
streamlit run app.py

## 📌 Note
Dataset and model files are not uploaded due to size limitations.

## 🎯 Output
- Predicts REAL or FAKE news
- Shows confidence score
